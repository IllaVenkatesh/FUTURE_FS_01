## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/your-username/FUTURE_FS_01.git
```

Navigate into the project

```bash
cd FUTURE_FS_01
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```

Preview production build

```bash
npm run preview
```
